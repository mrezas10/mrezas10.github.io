    Hatom Template Project

    Introduction

    The task is to integrate the web page using ReactJs library, and it ispermitted to use any external package such as (Lottie, SwiperJs, GSAP, etc) to ship the code quickly.

    Instructions

    ● Clone the page exactly as it apears in the template ● The coin must rotate as we scroll and lands on the middle of section 2. To get a better understanding, check out this demo. https://beta.hatom.com/hatom-token ● Submitting the newsletter form is not recommended unless the email address is validated

    Evaluation criteria

    ● the code will be evaluated based on code structure, programming best practices, responsiveness, and code quality. ● The final result must be pixel-perfect close to design template
